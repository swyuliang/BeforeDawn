# BeforeDawn
<ul>
    <li>游戏名称：黎明之前</li>
    <li>游戏类型：第一人称射击</li>
    <li>版本：Unity5.2.1</li>

    <li>怪物的自动巡逻，追踪目标。</li>
    <li>简单的剧情过场，对话交互。</li>
